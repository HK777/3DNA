##Как запускать приложение?
* Запустите скрипт [start.sh](start.sh)
* Ожидайте запуска прилежения должна появиться строчка
```
****-**-** **:**:**.***  INFO 37620 --- [           main] ru.dna.dna.Application                   : Started Application in * seconds (JVM running for *)
```
* перейдите на сайт [http://localhost:8080](http://localhost:8080)
